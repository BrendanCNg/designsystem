## React Usage

```jsx
import { Callout, CalloutCalendar } from './callout';

<Callout description="Description for this callout">
	Callout content
</Callout>

<CalloutCalendar
	description="description for this callout"
	datetime="2017-01-01T00:00:00+00:00"
	time="Sunday 32 Jun"
	subline="Your next appointment is"
	name="Talk to boss"
/>
```
